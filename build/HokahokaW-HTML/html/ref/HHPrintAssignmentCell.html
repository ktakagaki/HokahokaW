HOKAHOKA W SYMBOL

HHPrintAssignmentCell


HHPrintAssignmentCell prints the given symbols and their value assignements in an evaluatable cell.

DetailsDetails

  • Use to record value assignments in a notebook: the resulting assignment cell can then be run when re-opening the notebook with a fresh kernel.
