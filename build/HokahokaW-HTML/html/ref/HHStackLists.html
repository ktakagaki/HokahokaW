HOKAHOKA W SYMBOL

HHStackLists


HHStackLists takes lists and stacks the values (e.g. for stacked list plots with HHListLineStackPlot[])

DetailsDetails

  • Lists may be in one of the two following forms:
  • { {a1, a2, a3, ....}, {b1, b2, b3, ....}, ....}
  • { {t1, a1}, {t2, a2}, {t3, a3}, ....}, {{t1, b1}, {t2, b2}, {t3, b3}, ....}, ....}
